# 1. Introduction

This Brand Guide defines the official visual identity, tone, messaging, and design standards for **Yuhu**, ensuring consistency across all marketing, app UI, campus posters, social media, and developer contributions.

Yuhu’s brand must communicate:

* **Clarity**
* **Trust**
* **Modern campus culture**
* **Transparency and verification**
* **Youthful energy**

This guide should be followed by designers, developers, ambassadors, and marketing teams.

# 3. Logo Usage

## 3.1 Logo Variants

* Full Logo (Yuhu logotype + icon)
* Icon Only (for app launcher)
* White variant (for dark backgrounds)
* Dark variant (for light backgrounds)

## 3.2 Clear Space

Leave spacing equal to **height of the "Y"** around all sides.

## 3.3 DO NOTs

* Do not stretch or distort
* Do not rotate
* Do not change colors outside the palette
* Do not add shadows or outlines
* Do not place on cluttered backgrounds

## 3.4 Minimum Sizes

* App icon: 1024 × 1024 px
* Logo on posters: 200 px width minimum

# 5. Typography

Simple, modern, and readable across mobile screens.

## 5.1 Primary Font

**Inter** (Google Fonts)

* Headers: SemiBold / Bold
* Body: Regular / Medium

## 5.2 Font Sizes

* H1: 28–32 pt
* H2: 22–26 pt
* H3: 18–20 pt
* Body: 14–16 pt
* Caption: 12 pt

## 5.3 Line Height

* Body text: 1.3–1.5x

---

# 6. Iconography

* Use simple, outline-style icons
* Prefer Lucide Icons / Feather Icons
* Maintain consistent stroke width

**Examples:**

* Megaphone → Announcements
* Vote Box → Elections
* Users → Clubs
* Chat Bubble → Communities
* Bell → Notifications

# 9. UI Branding Rules

## 9.1 App Theme

* Default theme: **Dark Mode** (modern + reduces eye strain)
* Optional Light Mode toggle

## 9.2 Buttons

* Primary Button: Yuhu Blue (#295CFF)
* Secondary: Gray (#1E293B)

## 9.3 Cards

* Rounded corners: 12–16 px
* Shadows minimal or none
* Padding: 16–20 px

---

# 10. Marketing Material Templates

The Yuhu marketing team should use a consistent design language.

## 10.1 Posters

Required:

* Logo at top
* Verified badge element
* Clear title
* QR code to install
* Consistent margins

## 10.2 Instagram Posts / Reels

* High contrast
* Minimal text
* Large icons
* Strong visuals of students/clubs

## 10.3 Event Banners

Should include:

* Club name
* Event title
* Date & Time
* "Officially on Yuhu" label
